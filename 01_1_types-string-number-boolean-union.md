
# Types: String, Number, Boolean and Union Types

## String

```ts
let name: string;
name = 'Peter';
```

## Number

Including integers and decimals
```ts
let age: number;
age = 33;
```

## Boolean

```ts
let completed: boolean;
completed = false;
```

**Union Types**
When more than one type can be used

```ts
let zipCode: number | string;
zipCode = 94115;
zipCode = '94115';
```
